Max-Cut Benchmark Project

Solving Optimisation Problems Using Classical and Quantum-Inspired Methods

This project benchmarks Simulated Annealing (SA), Simulated Quantum Annealing (SQA), CP-SAT, and Gurobi on the Max-Cut problem using Erdős–Rényi graphs, comparing solution quality and runtime across a range of graph sizes and densities.
